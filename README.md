# 🚀 Perjalanan Belajar Frontend Development

## 📌 Tentang Repository Ini
Repository ini adalah dokumentasi nyata perjalanan belajar intensif saya selama 93 hari: dari titik nol hingga menjadi Junior Frontend Developer yang siap bersaing di tingkat internasional. 

Ini bukan sekadar tempat menyimpan kode, melainkan jurnal dari proses *deliberate practice* harian saya. Di sini, saya membangun *mental model* yang kuat tentang cara web bekerja di belakang layar, memahami fundamental sebelum bergantung pada *framework*, dan memastikan setiap baris kode yang saya tulis memiliki tujuan.

**Dimulai:** 23 Juni 2026  
**Target Selesai:** 24 September 2026

1. **Paham Konsep > Hafal Sintaks:** Mengutamakan pemahaman *Critical Rendering Path*, arsitektur browser, dan fondasi DOM sebelum menyentuh *tools* tingkat lanjut.
2. **Konsistensi Harian:** Menyelesaikan blok belajar harian, eksperimen dengan DevTools, dan *push* hasil belajar ke GitHub setiap hari.
3. **Active Recall:** Menggunakan Anki *flashcards* setiap hari untuk memastikan konsep yang dipelajari menempel secara permanen (jangka panjang).
